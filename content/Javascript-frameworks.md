# JavaScript Frameworks

Modern frontend frameworks provide component-based architectures for building interactive UIs.

## React

Developed by Meta. Uses a virtual DOM and JSX syntax. The most widely adopted framework with a massive ecosystem.

## Vue

A progressive framework that is easy to pick up. Offers a template-based syntax and reactive data binding.

## Svelte

A compiler-based approach that shifts work to build time, resulting in smaller bundles and faster runtime performance.
