# Python Data Structures

Python provides several built-in data structures that are essential for everyday programming.

## Lists

Ordered, mutable sequences.

```python
fruits = ["apple", "banana", "cherry"]
```

## Dictionaries

Key-value pairs with fast lookup.

```python
user = {"name": "Alice", "age": 30}
```

## Sets

Unordered collections of unique elements, useful for membership testing and deduplication.

```python
tags = {"python", "programming", "data"}
```
