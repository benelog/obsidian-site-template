# Python Web Frameworks

Python has a rich ecosystem of web frameworks for building applications of all sizes.

## Django

A batteries-included framework that provides ORM, admin panel, and authentication out of the box. Best suited for large, content-driven applications.

## Flask

A lightweight micro-framework that gives you flexibility to choose your own components. Great for APIs and smaller projects.

## FastAPI

A modern framework built on type hints and async support. Generates OpenAPI docs automatically and delivers high performance.
